HTML

Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser.

CSS

Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written
in a markup language such as HTML or XML (including XML dialects such as SVG, MathML or XHTML).

JAVASCRIP

JavaScript, often abbreviated as JS, is a programming language and core technology of the Web, alongside HTML and CSS 99% of
websites use JavaScript on the client side for webpage behavior.
